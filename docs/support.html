Why support helps

This repo is an open educational directory. Support helps maintain data freshness, improve taxonomy coverage, and expand the set of official engineering and AI study resources.

Ways to help

  • submit pull requests with validated official links
  • share the repo with engineers, students, and AI practitioners
  • support ongoing maintenance via Stripe if you want to fund more open learning repos
